# YUI LooP

琉球大学の学内移動の課題を解決するWebアプリ。

## プロダクトURL

- http://yui-loop.st.ie.u-ryukyu.ac.jp/

## 開発環境

- Node.js : 16.18.0-alpine

- Docker : 3.9

## 環境構築

```$ git clone https://github.com/RyunosukeFunasako/yui-loop.git```

```$ cd hfmy```

```$ docker-compose up```

でコンテナを起動。
